A specialized calculator designed to determine the optimal ratio of accumulators to solar panels for Factorio power networks. It helps players plan day/night energy storage requirements efficiently.

## Features

* **Ratio Calculation**: Instantly calculates the required number of accumulators based on solar panel count.
* **Power Analysis**: Provides detailed stats on power generation and storage capacity.
* **Day/Night Cycle**: Accounts for Factorio's specific day/night cycle timing.

## Installation and Usage

1.  Clone the repository or download the source files.
2.  Open `index.html` in any modern web browser.
3.  Enter the number of solar panels to see the requirements.
